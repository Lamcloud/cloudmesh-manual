cloudmesh.common.DateTime module¶

class cloudmesh.common.DateTime.DateTime[source]¶

Bases: object

This class provides some simple date time functions so we can use all the same format. Here is a simple example

start = DateTime.now() stop = DateTime.now() + DateTime.delta(1)

print (“START”, start) print (“STOP”, stop) print(“HUMANIZE STOP”, DateTime.humanize(stop - start)) print (“LOCAL”, DateTime.local(start)) print(“UTC”, DateTime.utc(start)) print(“NATURAL”, DateTime.natural(start)) print(“WORDS”, DateTime.words(start)) print(“TIMEZONE”, DateTime.timezone)

This will result in

START 2019-08-03 21:34:14.019147 STOP 2019-08-03 21:34:15.019150 HUMANIZE STOP a second ago LOCAL 2019-08-03 17:34:14 EST UTC 2019-08-03 21:34:14.019147 UTC NATURAL 2019-08-03 21:34:14.019147 UTC WORDS Sat 6 Aug 2019, 21:34:14 UTC TIMEZONE EST

class cloudmesh.common.DictList.DictList(entries=None, key='name', position='x')[source]¶

Bases: dict

A class to convert lists of dicts to dicts.

Example:

data = [ {“name”: “vm1”, “status”: “on”}, {“name”: “vm2”, “status”: “on”}, {“name”: “vm3”, “status”: “on”}, {“name”: “vm4”, “status”: “on”}, ]

d = DictList(data) pprint (d)

# {‘vm1’: {‘name’: ‘vm1’, ‘status’: ‘on’, ‘x’: 0}, # ‘vm2’: {‘name’: ‘vm2’, ‘status’: ‘on’, ‘x’: 1}, # ‘vm3’: {‘name’: ‘vm3’, ‘status’: ‘on’, ‘x’: 2}, # ‘vm4’: {‘name’: ‘vm4’, ‘status’: ‘on’, ‘x’: 3}}

print (d[‘vm1’]) # {‘name’: ‘vm1’, ‘status’: ‘on’, ‘x’: 0} # print (d.list()) # [{‘name’: ‘vm1’, ‘status’: ‘on’, ‘x’: 0}, # {‘name’: ‘vm2’, ‘status’: ‘on’, ‘x’: 1}, # {‘name’: ‘vm3’, ‘status’: ‘on’, ‘x’: 2}, # {‘name’: ‘vm4’, ‘status’: ‘on’, ‘x’: 3}]

One can switch the color mode off with:

Console.color = False
