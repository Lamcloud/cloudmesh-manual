abstract list()[source]¶

This command list all volumes as follows:

If NAMES are given, search through all the active clouds and list all the volumes. If NAMES and cloud are given, list all volumes under the cloud. If cloud is given, list all the volumes under the cloud. If cloud is not given, list all the volumes under current cloud. If vm is given, under the current cloud, list all the volumes attaching to the vm. If region is given, under the current cloud, list all volumes in that region.

Parameters

  • NAMES – List of volume names

  • vm – The name of the virtual machine

  • region – The name of the region

  • cloud – The name of the cloud

  • refresh – If refresh the information is taken from the cloud
